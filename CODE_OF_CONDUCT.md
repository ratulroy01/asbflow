# Code of Conduct

## Our Pledge

We pledge to make participation in this project a harassment-free experience for everyone.

## Our Standards

Examples of behavior that contributes to a positive environment:
- Demonstrating empathy and respect.
- Accepting constructive feedback gracefully.
- Focusing on what is best for the community.

Examples of unacceptable behavior:
- Harassment, insulting language, or discriminatory jokes.
- Publishing private information without explicit permission.
- Any conduct inappropriate in a professional setting.

## Enforcement

Project maintainers are responsible for clarifying and enforcing standards of acceptable behavior.

## Scope

This Code of Conduct applies within all project spaces and in public spaces when an individual is representing the project.

## Reporting

Report incidents privately to project maintainers.
